# AI Backend Engine

Enterprise-grade cloud-native backend built with FastAPI, PostgreSQL,
Redis, pgvector, and deployed via Docker + GitHub Actions CI/CD.

## Architecture
- **API Layer**: FastAPI with async Python
- **Data Layer**: PostgreSQL with indexing + pgvector for AI search  
- **Cache Layer**: Redis with cache-aside pattern
- **ML Layer**: Vector search + RAG pipeline
- **DevOps**: Docker Compose + GitHub Actions CI/CD

## Tech Stack
| Layer | Technology |
|-------|-----------|
| API | FastAPI, Pydantic |
| Database | PostgreSQL, SQLAlchemy |
| Cache | Redis |
| AI/ML | pgvector, LangChain |
| DevOps | Docker, GitHub Actions |
